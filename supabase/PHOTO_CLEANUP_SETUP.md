# حذف صور الطلبات تلقائياً بعد أسبوعين — دليل التشغيل

هذه الميزة تحذف صور الطلبات والحجوزات (اللي ترفعها من جهازك أو يرفعها العميل)
تلقائياً بعد **14 يوماً** من تاريخ رفعها، من التخزين ومن قاعدة البيانات معاً.

⚠️ **لم أقدر أنشرها مباشرة على حساب Supabase الخاص بك** لأني ما أملك صلاحية
الدخول على مشروعك (لا يوجد رابط أو مفتاح وصول لديّ). كل الأكواد جاهزة 100%،
تحتاج فقط تنفّذ الخطوات التالية مرة واحدة (تاخذ 5 دقائق):

## الخطوات

### 1) شغّل ملفات الهجرة (SQL)
افتح **Supabase Dashboard → SQL Editor → New query**، والصق محتوى كل ملف
بالترتيب وشغّله:

1. `supabase/migrations/004_storage_buckets_and_photo_cleanup.sql`
   (ينشئ مساحات التخزين الناقصة + أعمدة تتبع الحذف)
2. `supabase/migrations/005_schedule_photo_cleanup.sql`
   **لكن أولاً**: عدّل السطرين التاليين بداخله بقيمك الحقيقية قبل التشغيل
   (التفاصيل بالخطوة 3 أدناه):
   - `<YOUR-PROJECT-REF>`
   - `<YOUR-CLEANUP-SECRET>`

### 2) انشر الدالة السحابية (Edge Function)
من جهازك، داخل مجلد المشروع:

```bash
npx supabase login
npx supabase link --project-ref <YOUR-PROJECT-REF>
npx supabase functions deploy cleanup-old-photos
```

### 3) اضبط سرّ الحماية
اختر أي نص عشوائي طويل (مثلاً 32 حرف عشوائي) — هذا يمنع أي شخص من استدعاء
دالة الحذف من الإنترنت مباشرة:

```bash
npx supabase secrets set CLEANUP_SECRET=ضع-نص-عشوائي-طويل-وسري-هنا
```

ثم ضع **نفس القيمة** بدل `<YOUR-CLEANUP-SECRET>` داخل ملف
`005_schedule_photo_cleanup.sql` قبل تشغيله بالخطوة 1.

`<YOUR-PROJECT-REF>` تلقاه في: **Settings → General → Reference ID** بلوحة تحكم Supabase.

### 4) تأكد إنها تشتغل (اختياري)
```bash
curl -X POST "https://<YOUR-PROJECT-REF>.functions.supabase.co/cleanup-old-photos" \
  -H "x-cleanup-secret: نفس-السر-اللي-ضبطته"
```
لازم يرجع لك رد JSON فيه `"ok": true` وعدد الصور المحذوفة.

## كيف تشتغل الميزة بعد كذا؟
- تعمل تلقائياً **كل يوم الساعة 6 صباحاً بتوقيت السعودية** بدون أي تدخل منك.
- تحذف فقط صور الطلبات/الحجوزات التي مرّ على إنشائها أكثر من 14 يوماً.
- تفرّغ عمود الصور بقاعدة البيانات وتسجّل وقت الحذف (`photos_cleared_at`)
  حتى يبقى أثر واضح بالسجل.
- **باقي بيانات الطلب (الاسم، الحالة، السعر...) ما تتأثر إطلاقاً** — فقط الصور تُحذف.
